# JS Document Parser to CSV

Simple JavaScript tool to parse structured data from text documents and output CSV-style data.

## Features
- Reads text-based delivery documents
- Extracts key fields (date, product, quantity, reference)
- Outputs structured CSV-like data
- Clean and simple JavaScript logic

## Technologies
- JavaScript (Node.js)

## How to run

1. Make sure Node.js is installed
2. Edit the input document inside `sample-document.txt`
3. Run:

```bash
node index.js
